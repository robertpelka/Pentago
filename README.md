# Pentago

This program uses the GTK+ 3.18 library.
It can be compiled with the command `make` or ```gcc -std=c99 -Wall -Werror -o pentago main.c pentago.c gamestatus.c linfifo.c `pkg-config gtk+-3.0 --cflags --libs` ```
